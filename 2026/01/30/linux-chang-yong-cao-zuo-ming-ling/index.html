Linux常用操作命令

Linux 常用命令手册

本文档整理 Linux 日常运维、开发必备常用命令，按功能分类，包含 语法、示例、核心说明，兼顾新手入门与日常查阅，所有命令均经过实际测试（基于 CentOS/Ubuntu 通用版本）。

一、基础操作命令（必记）

此类命令用于 Linux 系统基础交互，高频使用，记熟可大幅提升操作效率。

1. 查看当前目录：pwd

  • 语法：pwd

  • 功能：Print Working Directory，显示当前所在的绝对路径。

  • 示例：
    `[root@localhost ~]# pwd

/root`

2. 切换目录：cd

  • 语法：cd [目录路径]

  • 功能：Change Directory，切换到指定目录。

  • 常用参数/示例：
    `cd /home # 切换到 /home 绝对路径目录

cd test # 切换到当前目录下的 test 子目录（相对路径）
cd .. # 切换到上一级目录
cd ~ # 切换到当前用户的家目录（root 是 /root，普通用户是 /home/用户名）
cd - # 切换到上一次所在的目录`

3. 查看目录内容：ls

  • 语法：ls [选项] [目录路径]

  • 功能：List，显示目录下的文件和子目录。

  • 常用选项：

    • -l：以长格式显示（包含权限、大小、修改时间等详细信息），简称「ll」（Linux 可自定义别名）

    • -a：显示所有文件/目录，包括隐藏文件（以 . 开头的文件）

    • -h：以人性化格式显示文件大小（如 K、M、G，替代默认字节）

  • 示例：
    `ls -l /home # 长格式显示 /home 目录内容

ll -ah # 人性化显示当前目录所有文件（包括隐藏文件）
ls -l | grep txt # 筛选显示当前目录下所有 .txt 结尾的文件`

4. 清屏：clear

  • 语法：clear 或 Ctrl + L（快捷键，更便捷）

  • 功能：清除终端屏幕上的所有内容，光标回到顶部。

5. 退出终端：exit

  • 语法：exit

  • 功能：退出当前终端会话（远程连接时断开连接，本地终端时关闭窗口）。

6. 查看命令帮助：man/–help

  • 语法：man [命令] 或 命令 –help

  • 功能：查看命令的详细用法（man 是手册，更全面；–help 是简易帮助）。

  • 示例：
    `man ls # 查看 ls 命令的完整手册（按 q 退出手册）

cd –help # 查看 cd 命令的简易帮助`

二、文件管理命令（高频）

用于文件/目录的创建、删除、复制、移动、查看等操作，是日常使用最多的命令组。

1. 创建文件：touch

  • 语法：touch [文件名]

  • 功能：创建一个空文件；若文件已存在，更新文件的修改时间。

  • 示例：
    `touch test.txt # 创建一个名为 test.txt 的空文件

touch a.txt b.txt # 同时创建 a.txt 和 b.txt 两个文件`

2. 创建目录：mkdir

  • 语法：mkdir [选项] [目录名]

  • 功能：Make Directory，创建新目录。

  • 常用选项：-p：递归创建多级目录（父目录不存在时自动创建）。

  • 示例：
    `mkdir test # 创建当前目录下的 test 目录

mkdir -p /home/test/abc # 递归创建 /home/test/abc 目录（即使 test 目录不存在）`

3. 删除文件/目录：rm

  • 语法：rm [选项] [文件/目录]

  • 功能：Remove，删除指定文件或目录（注意：Linux 中 rm 删除的文件无法恢复，慎用！）。

  • 常用选项：

    • -f：强制删除，不提示（即使文件不存在也不报错）

    • -r：递归删除，用于删除目录（删除目录及其下所有内容）

  • 示例：
    `rm test.txt # 删除 test.txt 文件（会提示确认，输入 y 确认删除）

rm -f a.txt # 强制删除 a.txt 文件，不提示
rm -rf test # 强制递归删除 test 目录及其下所有内容（高危命令，慎用）
rm -rf /home/* # 强制删除 /home 目录下所有内容（极度高危，禁止随意使用）`

4. 复制文件/目录：cp

  • 语法：cp [选项] 源文件/目录 目标路径

  • 功能：Copy，将源文件/目录复制到指定目标路径。

  • 常用选项：

    • -r：递归复制，用于复制目录（复制目录及其下所有内容）

    • -f：强制复制，覆盖目标路径下已存在的同名文件，不提示

    • -p：保留源文件的权限、修改时间等属性

  • 示例：
    `cp test.txt /home # 将当前目录的 test.txt 复制到 /home 目录下

cp -r test /home # 将当前目录的 test 目录复制到 /home 目录下
cp -rf a.txt /home # 强制将 a.txt 复制到 /home，覆盖同名文件`

5. 移动文件/目录：mv

  • 语法：mv [选项] 源文件/目录 目标路径

  • 功能：Move，将源文件/目录移动到指定目标路径；也可用于重命名文件/目录（目标路径与源路径同目录时）。

  • 常用选项：-f：强制移动，覆盖目标路径下同名文件，不提示。

  • 示例：
    `mv test.txt /home # 将当前目录的 test.txt 移动到 /home 目录下

mv test test_new # 将当前目录的 test 目录重命名为 test_new
mv -f a.txt b.txt # 强制将 a.txt 重命名为 b.txt（覆盖已存在的 b.txt）`

6. 查看文件内容：cat/tail/head/more

不同命令适用于不同场景（查看完整内容、查看末尾/开头、分页查看），按需选择。

（1）cat：查看文件完整内容（适合小文件）

（2）tail：查看文件末尾内容（适合日志文件）

（3）head：查看文件开头内容

（4）more：分页查看文件内容（适合大文件）

7. 搜索文件内容：grep

  • 语法：grep [选项] 搜索关键词 文件/目录

  • 功能：在文件或目录中搜索包含指定关键词的内容，是 Linux 中最常用的搜索命令。

  • 常用选项：

    • -i：忽略大小写（搜索时不区分大小写）

    • -n：显示匹配内容的行号

    • -r：递归搜索（搜索目录及其下所有文件）

    • -v：反向匹配（显示不包含关键词的行）

  • 示例：
    `grep “test” test.txt # 在 test.txt 中搜索包含 “test” 的内容

grep -in “hello” test.txt # 忽略大小写，搜索 “hello” 并显示行号
grep -r “root” /etc # 递归搜索 /etc 目录下所有包含 “root” 的文件
grep -v “注释” test.sh # 显示 test.sh 中不包含 “注释” 的行`

8. 查找文件/目录位置：find

  • 语法：find 搜索路径 [选项] 搜索条件

  • 功能：根据指定条件（文件名、大小、修改时间等）查找文件/目录的绝对路径。

  • 常用选项/条件：

    • -name 文件名：按文件名精确查找（支持通配符 *）

    • -size 大小：按文件大小查找（如 +10M：大于10M，-5K：小于5K）

    • -type f/d：按类型查找（f：文件，d：目录）

  • 示例：
    `find /home -name “test.txt” # 在 /home 目录下查找名为 test.txt 的文件

find / -name “*.sh” # 在整个系统中查找所有 .sh 结尾的文件
find /var -size +100M # 在 /var 目录下查找大于100M的文件
find /home -type d -name “test” # 在 /home 目录下查找名为 test 的目录`

三、系统管理命令（运维必备）

用于查看系统状态、进程管理、用户管理等，适合运维和开发排查问题。

1. 查看系统信息：uname/hostname

2. 查看系统负载：top/uptime

（1）top：实时查看系统负载和进程状态（类似 Windows 任务管理器）

（2）uptime：查看系统运行时间和负载（简易版 top）

3. 查看内存使用：free

4. 查看磁盘使用：df/du

（1）df：查看磁盘分区使用情况

（2）du：查看文件/目录占用磁盘大小

5. 进程管理：ps/kill

（1）ps：查看系统进程

（2）kill：终止指定进程

6. 用户管理：useradd/passwd/userdel

7. 权限管理：chmod/chown

Linux 中文件/目录的权限分为 读（r=4）、写（w=2）、执行（x=1），对应 所有者、所属组、其他用户 三类。

（1）chmod：修改文件/目录权限

（2）chown：修改文件/目录的所有者和所属组

四、网络操作命令（常用）

用于查看网络状态、测试网络连接、配置网络等，适合排查网络问题。

1. 查看网络配置：ifconfig/ip

2. 测试网络连接：ping

3. 查看端口占用：netstat/ss

4. 远程登录：ssh

5. 文件上传/下载：scp

五、其他常用命令（补充）

1. 压缩/解压命令：tar

2. 查看历史命令：history

3. 切换root用户：sudo

六、常用快捷键（提升效率）

  • Ctrl + L：清屏（等价于 clear 命令）

  • Ctrl + C：终止当前正在执行的命令（如 ping、top 等）

  • Ctrl + Z：暂停当前命令，后续可通过 fg 命令恢复

  • Tab 键：自动补全命令、文件名、目录名（按一次补全，按两次显示所有可选）

  • Ctrl + R：搜索历史命令（输入关键词，按 Enter 执行匹配的命令）

  • Ctrl + D：退出当前终端（等价于 exit 命令）

七、注意事项

  1. Linux 命令 严格区分大小写（如 ls 和 LS 是两个不同命令）。

  2. rm -rf 命令极度危险，删除的文件/目录无法恢复，执行前务必确认路径正确。

  3. 大部分命令需要 root 权限才能执行（普通用户可在命令前加 sudo）。

  4. 远程操作（ssh、scp）时，确保目标主机的对应端口（默认22）已开放，且网络连通。

  5. 若命令执行报错，可通过 命令 --help 或 man 命令查看正确用法，或搜索报错信息排查。
